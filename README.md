# 知乎标题关键词屏蔽脚本

一款轻量、纯净的知乎内容过滤油猴脚本，通过自定义关键词自动屏蔽标题包含指定内容的文章，打造清爽的浏览体验。

# ✨ 核心功能

- 自定义屏蔽关键词，标题包含关键词自动隐藏对应内容

- 支持推荐页、关注页、回答页全页面兼容

- 操作面板支持鼠标自由拖动、一键最小化，不遮挡页面内容

- 实时生效，添加/删除关键词无需刷新页面

- 关键词本地存储，关闭浏览器不丢失

- 极简设计，无冗余功能，占用内存极低

# 🛠️ 安装使用

## 1\. 安装脚本管理器

先在浏览器安装 Tampermonkey（油猴）或 Violentmonkey 扩展

## 2\. 安装脚本

1. 打开油猴扩展，新建用户脚本

2. 删除默认代码，粘贴本项目完整脚本代码

3. 保存脚本，刷新知乎页面即可使用

## 3\. 基础操作

- 添加关键词：在输入框输入内容，按回车键即可添加

- 删除单个关键词：点击关键词右侧的 × 按钮

- 清空所有关键词：点击「清空所有关键词」按钮

- 移动面板：按住顶部灰色栏拖动

- 最小化面板：点击顶部 − 按钮，点击 □ 恢复

# 📁 项目说明

- 关键词匹配不区分大小写，采用包含匹配规则

- 关键词数据仅保存在当前浏览器本地，更换浏览器需重新添加

- 脚本仅作用于知乎网页端，不影响APP使用

- 请勿添加违法、违规、侵权相关关键词
